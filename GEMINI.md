Todas las respuestas, explicaciones y logs generados deben estar estrictamente en español.
Tienes permiso para ejecutar comandos de lectura (ls, cat, type), consultas de estado de red, creacion de carpetas y archivos y generar cambios menores sin preguntar.
No tienes permiso para eliminar carpetas o archivos o instalar dependencias sin autorizacion del usuario.
Cuando ejecutes npm start o npm run dev y el proceso sea persistente, confírmame que la sala está abierta y pregúntame si deseo liberar la terminal para seguir conversando.
Ten en cuenta que este es un proyecto que va a quedar "en desarrollo" asi que no te esfuerzes en completar funcionalidades, simplemente toca el apartado estetico.
Tenes prohibido tocar funcionalidades del backend, solo podés modificar LOS COMENTARIOS del backend.
